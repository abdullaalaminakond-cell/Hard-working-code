# 🐍 Boss Snake Game (with AI & Custom Music)

স্বাগতম! এটি আমার তৈরি একটি স্পেশাল এবং খতরনাক **Snake Game** যা আমি সম্পূর্ণ আমার মোবাইল ফোনে **Pyroid 3** অ্যাপ এবং **Pygame** লাইব্রেরি ব্যবহার করে কোড করেছি।

## ✨ গেমের দারুণ সব ফিচার (Features):
- **মাল্টিকালার সাপ:** সাপের বডি একঘেয়ে নয়, এটি লাল, নীল এবং সাদা রঙে পরিবর্তন হতে থাকে।
- **খতরনাক বস শিপ (Boss Ship):** গেমে শক্তিশালী বস এবং টেলিপোর্ট করা শত্রু সাপ আছে, যারা আপনাকে তাড়া করবে!
- **ম্যাজিক সার্কেল (Magic Circle):** স্ক্রিনে আসা ম্যাজিক সার্কেলে ঢুকলে সাপটি কিছু সময়ের জন্য অমর (Invincible) হয়ে যায়।
- **স্মার্ট মিউজিক সিস্টেম:** আপনার মোবাইলে গান থাকলে ব্যাকগ্রাউন্ডে চমৎকার MP3 গান বাজবে। আর গান না থাকলেও গেম ক্র্যাশ করবে না, একদম স্মুথলি চলবে!

## 🛠️ টেকনোলজি (Tech Stack):
- **Language:** Python 3
- **Library:** Pygame
- **Platform:** Android (Pyroid 3)

## 🚀 ভৱিষ্যতৰ আপডেট (Upcoming Updates):
- **টাইম লিমিট (Time Limit):** খুব সোনকালে গেমে থকা **Yellow Circle** আৰু **Blue Circle** ৰ বাবে এটা নিৰ্দিষ্ট সময়ৰ সীমা (Time Limit) লগাই দিয়া হ’ব।
- **নতুন শত্ৰুৰ আক্ৰমণ (Enemy Spawn):** গেমৰ **Boss Snake** আৰু **Friend Snake** সদায় স্ক্ৰীণৰ একেবাৰে ওপৰৰ (From the top) পৰা ওলাই আক্ৰমণ কৰিবলৈ আহিব!
- ## 🎵 মিউজিক আৰু এপিকে (Music & APK Note):
- **গান কিয় নুশুনিব?:** এই গেমৰ বেকগ্ৰাউণ্ড মিউজিক কেৱল মোৰ মোবাইলত থকা MP3 গানৰ ওপৰত কাম কৰে। গতিকে আপোনাৰ মোবাইলত গানবোৰ নাথাকিলে গেমটো মনে মনে চলিব (কোনো এৰৰ নাহে)।
- **কম্পিউটাৰ আৰু APK প্লেন:** মোৰ ওচৰত বৰ্তমান কম্পিউটাৰ/লেপটপ নাই। যদি থাকিলহেঁতেন, মই এই পাইথন ক’ডটোক এপিকে (APK) ফাইল বা বিল্ড জাৰ (.jar) ফাইললৈ কনভাৰ্ট কৰি দিলোঁহেঁতেন, যাতে সকলোৱে ডাইৰেক্ট গেমটো ইনষ্টল কৰিয়েই গানৰ সৈতে খেলিব পাৰিলেহেঁতেন!
